Dokumentasi Kubernetes

Kubernetes adalah sebuah orkestrator kontainer open source, yang mengotomasi deployment, replikasi, dan pengaturan aplikasi kontainer dengan mudah. Cloud Native Computing Foundation (CNCF) mewadahi proyek open source ini.
