Documentation de Kubernetes

Kubernetes est un moteur d'orchestration de conteneur open source permettant d'automatiser le déploiement, la mise à l'échelle et la gestion des applications conteneurisées. Le projet open source est hébergé par la Cloud Native Computing Foundation. (CNCF).
